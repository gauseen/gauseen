从 Chrome 的开发者工具调试 Android 微信 Web 网页

发表于 2018-12-08 | 原作者 gauseen |

具体步骤：

  1. 在手机微信内，打开网址 http://debugx5.qq.com
  2. 信息Tab下 –> TBS settings –>　勾选【打开 TBS 内核 Inspector 调试功能】（根据自己需求选择其它）
  3. 微信提示需要重启，点击重启
  4. 完成上述步骤后，打开Android手机USB调试功能
  5. 用 USB 数据线连接手机与电脑
  6. 选择手机USB连接方式为：传输文件，等待驱动安装完毕
  7. 用 Chrome 浏览器打开 chrome://inspect/#devices
  8. 这样就可以在Chrome浏览器里，调试移动端微信内的Web页面了

注： 该功能需要翻墙才可使用
