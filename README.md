### 使用步骤

#### 克隆代码库
``` bash
git clone https://github.com/gauseen/gauseen.github.io.git
```

#### 安装依赖资源
```sh
yarn install
```

#### 清空 hexo 构建文件
```sh
yarn clean
```

#### hexo 构建项目
```sh
yarn generate
```

#### 本地运行代码
```sh
yarn dev
```

#### 部署到 github
```sh
yarn deploy
```

#### 新增页面
```sh
hexo new <title>
```
